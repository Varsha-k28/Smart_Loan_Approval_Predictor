About Loan Approval Predictor

The Loan Approval Predictor is an Artificial Intelligence powered web application developed using Machine Learning, Flask and Bootstrap.

The prediction model analyzes an applicant's financial information including income, loan amount, CIBIL score, assets and employment details to determine whether a loan application is likely to be approved.
